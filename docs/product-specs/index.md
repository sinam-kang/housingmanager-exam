# Product Specs — 주택관리사 기출문제 앱

## 앱 개요
주택관리사 기출문제 PDF를 임포트하여 학습하는 개인용 도구.

## 사용자
단독 사용자 (본인 전용)

## 데이터 소스
연도별 폴더 구조의 PDF 파일 (문제/정답 파일 분리, 해설 없음)

- 1차 시험: 1교시(민법, 회계) / 2교시(시설개론)
- 2차 시험: 관계법규, 공동주택관리실무 (객관식+주관식 혼합)

## 핵심 설계 결정
- 단원 태깅: 임포트 시 LLM이 문제 내용 기반으로 자동 분류
- 문제-정답 매칭: 문제 번호 기반 자동 매칭
- 해설: PDF에 없음 → 오답노트에서 "해설 보기" 버튼 누를 때 LLM이 생성 및 저장

## 기능 흐름
```
퀴즈 모드에서 오답 → 오답노트에 자동 저장 → 오답노트에서 "해설 보기" → LLM 해설 생성 → 저장
```

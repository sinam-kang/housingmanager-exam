# 필터

공통 필터 컴포넌트로 문제 조회, 퀴즈 모드 시작, 오답노트에서 공통 사용.

## 필터 항목

### 연도
- 임포트된 연도 목록에서 다중 선택 가능
- 기본값: 전체

### 시험차수
- 1차 / 2차 선택
- 기본값: 전체

### 과목
- 시험차수 선택 시 해당 과목만 표시
  - 1차: 민법 / 회계 / 시설개론
  - 2차: 관계법규 / 공동주택관리실무
- 다중 선택 가능
- 기본값: 전체

### 단원 (계층형)
- 과목 선택 후 해당 과목의 단원 목록만 표시
- 과목 미선택 시 단원 필터 비활성화
- 다중 선택 가능
- 기본값: 전체

### 문제 유형
- 객관식 / 주관식
- 기본값: 전체

### 기타
- 북마크한 문제만 보기 (토글)
- 오답 이력 있는 문제만 보기 (토글)

## 필터 동작 규칙
- 모든 필터는 AND 조건으로 적용
- 필터 초기화 버튼으로 전체 기본값으로 복원
- 적용된 필터 수를 필터 버튼에 뱃지로 표시 (예: 필터 3)

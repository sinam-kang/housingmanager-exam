# 퀴즈 모드

## 퀴즈 시작 설정

### 범위 선택 (복수 조합 가능)
| 범위 유형 | 설명 |
|----------|------|
| 연도/과목/단원 | 필터 조합으로 범위 지정 |
| 북마크 문제만 | 북마크된 문제만 출제 |
| 오답 문제만 | 오답 이력이 있는 문제만 출제 |
| 전체 랜덤 N문제 | 전체 문제 중 N개 무작위 선택 |

### 출제 방식 (시작 전 선택)
- 순서대로: 문제 번호 오름차순
- 랜덤 셔플: 무작위 순서

## 퀴즈 진행 화면
- 문제 텍스트 표시
- 진행 상황 표시 (예: 3 / 20)
- 객관식: 보기 ①②③④⑤ 선택
- 주관식: 정답 공개 후 사용자가 "맞음 / 틀림" 직접 선택
- 답 제출 시 즉시 맞음/틀림 피드백 표시
- 틀린 경우 자동으로 오답노트에 저장

## 퀴즈 종료 화면
- 총 문제 수 / 맞은 수 / 틀린 수 / 정답률
- 오답 문제 목록 (바로 오답노트로 이동 가능)
- 다시 풀기 / 홈으로 버튼

## 오답 처리
- 틀린 문제는 즉시 오답노트에 저장
- 같은 문제를 여러 번 틀려도 오답노트에 중복 저장하지 않고 틀린 횟수만 누적
- 해설은 오답노트에서 별도로 요청 (→ wrong-answer-notes.md)

# 북마크/즐겨찾기

## 북마크 태그
문제마다 아래 3가지 태그 중 하나를 선택하여 북마크.

| 태그 | 의미 |
|------|------|
| 중요 | 반드시 다시 볼 문제 |
| 어려움 | 이해가 안 되는 문제 |
| 나중에 | 일단 넘기고 나중에 볼 문제 |

- 문제 1개에 태그 1개만 가능
- 태그 변경 가능 (다른 태그 선택 시 교체)
- 북마크 해제 가능 (태그 제거)

## 북마크 등록
- 문제 목록 / 문제 상세 / 퀴즈 진행 중 / 오답노트 어디서든 북마크 가능
- 북마크 버튼 클릭 시 태그 선택 팝업 표시

## 북마크 목록 화면
- 태그별 탭으로 구분 (전체 / 중요 / 어려움 / 나중에)
- 각 항목: 연도 / 과목 / 문제번호 + 문제 텍스트 미리보기
- 클릭 시 문제 상세 화면으로 이동
- 필터 및 키워드 검색 적용 가능 (→ filters.md)

## 퀴즈 연동
- 북마크 목록에서 "이 목록으로 퀴즈 시작" 버튼 제공
- 태그별로 퀴즈 시작 가능 (예: "어려움" 태그만 퀴즈)

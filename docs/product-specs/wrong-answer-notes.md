# 오답노트 + 해설 보기

## 오답노트 목록 화면
- 오답 이력이 있는 문제들을 리스트로 표시
- 각 항목에 표시하는 정보:
  - 연도 / 과목 / 단원
  - 문제 번호 + 문제 텍스트 미리보기
  - 틀린 횟수
  - 해설 생성 여부 아이콘
  - 북마크 여부 아이콘

### 정렬 (사용자가 선택)
| 정렬 기준 | 설명 |
|----------|------|
| 최근 틀린 순 | 가장 최근에 틀린 문제가 위로 |
| 틀린 횟수 순 | 많이 틀린 문제가 위로 |
| 과목/단원별 | 과목 → 단원 그룹핑 |

## 오답 상세 화면
- 문제 전체 텍스트
- 보기 ①②③④⑤ (객관식만)
- 정답 표시
- 틀린 횟수 / 마지막으로 틀린 날짜
- "해설 보기" 버튼
- 북마크 버튼
- 오답노트에서 제거 버튼

## 해설 보기 (LLM 생성)
- "해설 보기" 버튼 클릭 시 LLM 호출
- 입력: 과목명 + 문제 텍스트 + 보기 + 정답
- 출력: 정답 이유 + 오답 보기별 틀린 이유 (객관식) / 정답 풀이 (주관식)
- 생성된 해설은 DB에 저장 → 이후 버튼 클릭 시 저장된 해설 즉시 표시 (재호출 없음)
- 해설 생성 실패 시 오류 메시지 표시 후 재시도 가능

## 오답 기록 규칙
- 같은 문제를 여러 번 틀려도 오답노트 항목은 1개 유지
- 틀린 횟수만 누적
- 퀴즈에서 맞혀도 오답노트에서 자동 제거하지 않음 (수동 제거만 가능)

# 문제 조회/검색

## 화면 구성

### 문제 목록 화면
- 문제들을 리스트로 나열
- 각 항목에 표시하는 정보:
  - 연도 / 시험차수 / 과목
  - 문제 번호 + 문제 텍스트 앞부분 미리보기 (2줄 이내)
  - 북마크 여부 아이콘
  - 오답 이력 여부 아이콘
- 클릭 시 문제 상세 화면으로 이동

### 문제 상세 화면
- 문제 전체 텍스트
- 보기 ①②③④⑤ (객관식만)
- "정답 보기" 토글 버튼 (기본 숨김)
- 북마크 버튼
- 퀴즈 모드로 이동 버튼 (해당 문제부터 시작)

## 검색
- 문제 텍스트 키워드 검색
- 검색과 필터 동시 적용 가능

## 필터 연동
- 연도 / 과목 / 단원 / 문제유형(객관식·주관식) 필터 적용 가능
- 필터 상세 스펙 → filters.md

## 정렬
- 기본: 연도 내림차순 → 문제번호 오름차순
- 변경 불필요 (단순 학습 도구이므로 고정)

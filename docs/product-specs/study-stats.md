# 학습 통계/진도

## 화면 구성
통계 대시보드 형태의 단일 화면. 섹션별로 구분하여 표시.

## 통계 항목

### 전체 진도
- 전체 문제 수 대비 풀어본 문제 수 및 비율 (%)
- 진행 바(progress bar)로 시각화

### 과목별 정답률
- 각 과목의 풀어본 문제 수 / 정답 수 / 정답률
- 막대 그래프로 시각화

### 연도별 정답률
- 각 연도의 풀어본 문제 수 / 정답 수 / 정답률
- 막대 그래프로 시각화

### 취약 단원
- 단원별 정답률 계산 후 낮은 순으로 상위 5개 표시
- 단원명 + 정답률 + 풀어본 문제 수 표시
- 해당 단원 클릭 시 필터 적용된 문제 목록으로 이동

### 최근 퀴즈 기록
- 날짜별 퀴즈 세션 이력 리스트
- 각 항목: 날짜 / 범위 / 문제 수 / 정답률
- 최근 30개까지 표시

### 연속 학습일
- 오늘까지 퀴즈를 푼 날이 며칠 연속인지 표시
- 최장 연속 학습일 기록도 함께 표시

## 통계 계산 기준
- 같은 문제를 여러 번 풀면 마지막 풀이 결과를 기준으로 정답률 계산
- 단, 퀴즈 기록은 세션 단위로 모두 저장

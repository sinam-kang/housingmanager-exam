# Product Sense

## 이 앱이 존재하는 이유
주택관리사 시험 준비를 위해 기출문제를 반복 학습하는 도구.
시중의 복잡한 플랫폼 없이, 본인이 가진 PDF 파일을 그대로 활용하여 효율적으로 공부할 수 있도록 한다.

## 핵심 사용자
- 주택관리사 자격증을 준비 중인 수험생 (본인 1명)
- 비개발자 — 기술적 설정 없이 바로 쓸 수 있어야 함

## 이 앱이 잘 되고 있다는 신호
- PDF 임포트 후 바로 문제를 풀 수 있다
- 오답노트를 통해 취약한 부분을 파악할 수 있다
- 어떤 단원이 부족한지 통계로 확인할 수 있다
- 앱 사용법을 배우는 데 시간을 쓰지 않아도 된다

## 이 앱이 잘못되고 있다는 신호
- PDF 파싱 오류가 잦아서 수동으로 수정해야 하는 경우가 많다
- UI가 복잡해서 원하는 기능을 찾기 어렵다
- 학습보다 앱 조작에 시간이 더 걸린다

## 의사결정 원칙
기능 추가나 설계 결정 시 아래 순서로 우선순위를 판단한다:

1. **학습 흐름을 방해하지 않는가** — 문제 풀기에 집중할 수 있어야 한다
2. **비개발자가 쉽게 쓸 수 있는가** — 설정, 설치, 실행이 단순해야 한다
3. **데이터가 안전한가** — 임포트한 문제와 학습 기록이 손실되지 않아야 한다

## 범위 밖
- 다중 사용자 지원
- 클라우드 동기화
- 문제 직접 편집/추가 기능 (PDF 임포트로 충분)
- 소셜 기능 (공유, 랭킹 등)
